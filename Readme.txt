InstaMétéo

Groupe : Steven ROBILLARD, Thomas RAINEAU, Emeric MOTTIER

Le site est accessible à partir du site web : www.thomasraineau.com/projets/ProjetMeteo

Api Insta
	Nous utilisons l’API d’Instagram afin de récupérer les photos des comptes recherchés. De plus nous récupérerons différentes informations sur ces photos comme par exemple le lieu où elle a été prise.

Api Météo
	L’API Météo que nous utilisons pour notre site web est Open Weather Map. Grâce à cette API nous récupérons la météo de la ville recherchée par l’utilisateur. 

lien : https://openweathermap.org/

Api Géo
	L’API Géolocalisation est celle de Google Maps. nous pourrons récupérer la position de l’utilisateur lorsque celui-ci accèdera au site web.

lien : https://developers.google.com/maps/documentation/javascript/examples/map-geolocation?hl=fr
https://developers.google.com/maps/documentation/geocoding/intro?hl=fr
